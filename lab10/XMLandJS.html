XML vs Java Script

  • What is XML?

Markup languages are languages that consist of directions that tell the software in a browser how to display and manage a web browser. XML is Extensible Markup Language. It is a text based syntax designed to describe, deliver, and exchange structured information.

  • How is XML used?

XML is used to extend the power of HTML by seperating data from presentation.

  • What is Javascript?

Javascript is an object based, client side scripting language interpreted by a web browser.

  • How is Javascript related to HTML and CSS?

HTML brings structure to a webpage and CSS adds style. When combined with HTML and CSS Javascript brings interactivity to a web page.
